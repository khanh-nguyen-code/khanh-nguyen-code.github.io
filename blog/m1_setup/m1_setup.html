How I install applications in M1 macbook




Package Manager

MacPorts - https://www.macports.org/
MacPorts principles emphasize building its little world without any dependency from the host OS. (opposite to Homebrew)


Script

Put all custom scripts into ~/bin


CLI Application

Preferably install from MacPorts, if the port is unavailable, request the authors. Otherwise, download binaries or compile the source code and put into ~/Packages folder.


Python

Latest python package name - python39
Install Python - sudo port install <python package name>
Set allias for python - sudo port select --set python <python package name>
Install python environment (miniforge3 - conda) - https://github.com/conda-forge/miniforge/releases


GUI Application

Preferably download .app file (similar to AppImage for Linux).
