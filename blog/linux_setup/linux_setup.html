How I install an application on Linux

First Rule

NEVER MESS WITH THE DEFAULT PACKAGE MANAGER. IT IS USED TO WORK WITH THE OPERATING SYSTEM ONLY!

GUI Application

(1) Find a distribution on flathub or AppImageHub and you are done.
(2) If the previous steps does not work. Try with linuxbrew, download the binaries or build the application from source.
(3) Create a .desktop file and put it in ~/.local/share/applications/ to create a GNOME application launcher.

CLI Application

(1) Try with linuxbrew, download the binaries or build the application from source.
(2) Edit PATH in ~/.bashrc to access the binaries globally.
