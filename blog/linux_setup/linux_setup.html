How I install an application on Linux

First Rule

NEVER MESS WITH THE DEFAULT PACKAGE MANAGER. IT IS USED TO WORK WITH THE OPERATING SYSTEM ONLY!

GUI Application

(1) If the application can be run on sandbox, find a distribution on flathub or AppImageHub and you are done.
(2) Download the binaries or build the application from source.
(3) Create a .desktop file and put it in ~/.local/share/applications/ to create a GNOME application launcher.

CLI Application

(1) Find the package on linuxbrew and you are done.
(2) Download the binaries or build the application from source.
(3) Edit PATH in ~/.bashrc to access the binaries globally.
