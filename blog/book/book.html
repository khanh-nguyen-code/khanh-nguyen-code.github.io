Textbook Recommendation

All of them here

Theoretical Computer Science

  • Introduction to the theory of computation - Michael Sipser - 1996
    (Prof Zinovi Rabinovich recommendation) the book covers most basic concepts about automata theory, computability theory and complexity theory with rigorous proofs
  • Arora, Sanjeev, and Boaz Barak. Computational complexity: a modern approach. Cambridge University Press, 2009
    to read
    (Prof Zinovi Rabinovich recommendation)

Algorithms

  • Introduction to algorithms - Thomas H Cormen, Charles E Leiserson, Ronald L Rivest, and Clifford Stein - 2009
    the famous CLRS book. this textbook covers many topics in algorithms with great details. Prof Zinovi Rabinovich once told us, this is the must have textbook for everyone.

Machine Learning

  • Machine learning: a probabilistic perspective - Kevin P Murphy - 2012
    "This books adopts the view that the best way to make machines that can learn from data is to use the tools of probability theory"
    as in the preface of the book, the book describes the correct approach to machine learning
  • Anzai, Yuichiro. Pattern recognition and machine learning. Elsevier, 2012
    to read

Linear Algebra I

Linear Algebra II

  • Linear algebra done right. Springer - Sheldon Axler - 2015
    the pure math textbook for linear algebra, built up from the concepts of vector space and invariant subspace to trace and determinant. the must read 2nd book in linear algebra

Real Analysis I

  • Understanding analysis - Stephen Abbott - 2012
    the pure math textbook for analysis I, all theorems are derived from a set of axioms of real set. the last chapter provides a construction of real set from rational set clear out all the doubts about the properties of real numbers. it is not a textbook, it is a movie

Abstract Algebra

  • Abstract algebra - Dummit, David Steven, and Richard M. Foote - 2004
    to read

ODE & PDE

  • Ordinary Differential Equations - William A. Adkins, Mark G. Davidson - 2012
    "this textbook gives an early presentation of the Laplace transform, which is then used to motivate and develop many of the remaining differential equation concepts for which it is particularly well suited"
    to read
  • Introduction to Partial Differential Equations - Peter J. Olver - 2014
    to read
